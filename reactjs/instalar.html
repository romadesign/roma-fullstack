# 1.- Primeros pasos

Vamos a comenzar con las instalaciones necesarias para configurar nuestro proyecto.

# Crear una nueva aplicación React

Solo se realiza 1 sola vez. Si esta en linux y te da error puedes utilizar "sudo"

        $ npm install -g create-react-app
        $ sudo npm install -g create-react-app

# Creando un proyecto React

        $ create-react-app nombre_del_proyecto
        $ cd nombre_del_proyecto
        $ npm start
