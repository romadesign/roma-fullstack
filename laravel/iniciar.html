# 1.- Primeros pasos

Crear nuestro proyecto utilizando Laravel.

# Crear repositorio

App laravel y ir al archivo creado

        composer create-project laravel/laravel app-test

# Arrancar proyecto y mirar en el navegado

        php artisan serve

# Autenticación con laravel

Primero realizamos una instalación con composer

        composer require laravel/ui

La opción de autenticación viene con vueJs instalado

        php artisan ui vue --auth

Leer el mensaje de la consola o realizar estas opciones para instalar los paquetes Node Modules

        npm install && npm run dev

# Activar la conexión para la Base de datos

Ir al archivo .env

        npm install && npm run dev

# Instalando Modulos principales

Instalando Flask y MongoDb, solo se instalara en nuestro proyecto por eso estamos utilizando un entorno virtual. https://flask-pymongo.readthedocs.io/en/latest/ (opens new window)

        $ pip install Flask-PyMongo
