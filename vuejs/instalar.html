# 1.- Primeros pasos

Vamos a comenzar con las instalaciones necesarias para configurar nuestro proyecto.

# Instalando Vue en el ordenador

Solo se realiza 1 sola vez

        $ npm install -g @vue/cli
        $ vue --version

# Creando proyecto Vue

Creando un nuevo proyecto, run

     $ vue create nombre_de_la_carpeta
